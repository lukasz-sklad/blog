# Brew nie tylko macos

Bardzo znane narzędzie z systemu macos instalujące oprogramowanie linux

## Instalacja w macos

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Usuwamy brew z systemu macos

Pojedyncze polecenie odinstalowania homebrew wygląda następująco, w zależności od wersji macos:

macos Catalina, macos Big Sur i macos Mojave:
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/uninstall.sh)"`

macos High Sierra, Sierra, El Capitan:
`ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/uninstall)"`

## Aktualizacja:

`brew update && brew upgrade`

ale brew można doskonale wykorzystywać nawet w systemie linux, brewlinux pozwala na używanie choćby dowolnej wersji oprogramowania. Inaczej mówiąc jest świetnym menadżerem do zarządzania oprogramowaniem

## Instalacja w linux

Instalujemy dodatkowe i niezbędne oprogramowanie dla Debian i pochodnych:

```
apt install build-essential procps curl file git
```

Przechodzimy do instalacji brew

```
curl -fsSL -o install.sh https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh
```

ściągnięty plik

`less install.sh` możemy przejrzeć

```
/bin/bash install.sh
```

## Ustawienie zmiennych:

```
echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> /home/lmk/.profile
eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"
```

Instalujemy przykładowe oprogramowanie:

```
brew update --auto-update
brew install moreutils
```
