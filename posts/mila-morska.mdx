<h1> 1 mila morska – obliczenia</h1>

Różne obliczenia widziałem na internecie...
ale my wolimy po swojemu.

<h3>Zacznijmy od równika ma 40 000 km</h3>

a pełny kąt to 360°

<h3>Dzielimy:</h3>

```
>>> 40000 / 360
111.11111111111111
>>> 
```
czyli 1° na równiku = 111.11111111111111 km ≈ 111.11 km
a 1° wiemy że ma 60'

<h3>I znowu dzielimy:</h3>

```
>>> 111.11111111111111 / 60
1.8518518518518519
>>>
```
czyli <strong>NM = 1852 m</strong>

lub obliczamy minuty kątowe

```
>>> 360 * 60
21600
>>>
```
1 NM = 40 000 km / 21 600

```
>>> 40000 / 21600
1.8518518518518519
>>>
```
hmm... kto by się spodziewał

<strong>NM = 1852 m</strong>

Mile możemy podzielić na kable, zgodnie z zasadą: 1 NM = 10 kabli, czyli Kabel (kbl) jest równy jednej dziesiątej mili morskiej.

Jeszcze ze znanych morskich jednostek mamy węzły:

1w = 1852m / 3600 s

```
>>> 1852 / 3600
0.5144444444444445
>>>
```
0.514 m/s ≈ 0.5 m/s
