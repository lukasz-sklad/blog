# Instalujemy hishtory

Instalujemy hishtory, co to jest jak to zjeść. Masz czasem problem, jakie ostatnio komendy wykonywałeś to narzędzie pomorze nam ogarnąć to. Po zainstalowaniu wydajemy polecenie ctr+r, pomocne a zobacz to teraz.

Wydajemy polecenie:

```
curl https://hishtory.dev/install.py | python3 -
```

Na innym komputerze instalujemy z naszym kluczem:

klucz możemy podejrzeć wydając polecenie:

```
hishtory status
```

```
curl https://hishtory.dev/install.py | python3 -
hishtory init $YOUR_HISHTORY_SECRET
```

Fajna sprawa!!!
