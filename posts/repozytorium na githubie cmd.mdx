# Zakładanie repozytorium na githubie konsola

Otwieramy **konsolę** w danym folderze i wykonujemy następujące polecenia:

```
echo "# abc-ABC-ab-01" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:lukasz-sklad/abc-ABC-ab-01.git
git push -u origin main
```
