# Zakładanie repozytorium na githubie konsola

Otwieramy **konsolę** w danym folderze i wykonujemy następujące polecenia:

Klonujemy puste repozytorium 

```
git clone git@github.com:lukasz-sklad/abc-ABC-ab-01.git
```
wchodzimy do naszego katalogu
cd abc-ABC-ab-01

w tym katalogu wykonuje poniższe polecenia:
ls -la 
zobaczymy tam tylko jeden plik o nazwie .git

```
echo "# abc-ABC-ab-01" >> README.md
```

```
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:lukasz-sklad/abc-ABC-ab-01.git
git push -u origin main
```
